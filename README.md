# Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.  
This project demonstrates basic game logic, DOM manipulation, and event handling.

## Features

- Two player mode
- Win detection
- Draw detection
- Restart game option
- Responsive layout

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

tic-tac-toe/
│
├── index.html  
├── style.css  
├── script.js  
└── images/

## How to Run

1. Clone the repository:
   git clone https://github.com/vivek-jadhav77/tic-tac-toe.git

2. Open the project folder.

3. Open `index.html` in your browser.
